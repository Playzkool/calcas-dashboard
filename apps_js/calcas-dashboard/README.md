# calcas-dashboard — Frontend

Application React (TypeScript + Vite + MUI).

Pour la documentation complète du projet (installation, données de démonstration,
identifiants), voir le [README racine](../../README.md).

## Commandes

```bash
npm install      # installer les dépendances
npm run dev      # serveur de développement (port 5173, proxy vers Django :8000)
npm run build    # build de production
npm run lint     # ESLint
```
